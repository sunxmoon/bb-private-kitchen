# 宝宝的私房菜馆 (Baby's Private Kitchen)

A lightweight, mobile-first web application for family members to order dishes and manage a shared dish library.

## Features
- Dish Management
- Family Member Ordering
- Order History and Audit Logs
- Mobile-responsive UI
- Preference memory (Fill "same as last time")

## Tech Stack
- FastAPI
- SQLAlchemy
- Jinja2 Templates
- Tailwind CSS
- Docker
